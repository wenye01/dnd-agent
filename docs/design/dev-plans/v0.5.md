# v0.5 开发计划 - 功能完善

> **版本**: v0.5
> **状态**: 计划中
> **前置版本**: v0.4 (法术+物品+地图)
> **目标版本**: v1.0 (完整版本)

---

## 1. 版本概述

### 1.1 版本目标

在 v0.4 基础上，完善 D&D 5e 规则覆盖，实现剧本完整流程，优化用户体验。

### 1.2 核心功能

| 功能领域 | 实现内容 |
|----------|----------|
| **剧本系统** | 剧本完整流程、章节管理、事件触发器 |
| **世界地图** | 冒险地图、地点切换、旅行系统 |
| **存档系统** | 多存档、加载、删除、自动保存 |
| **规则完善** | 专长（B304）、背景特性（B305）；死亡豁免基础在 v0.3 B210，完整版在 v1.0 B401 |
| **上下文优化** | 历史压缩、渐进式披露 |

### 1.3 发布标准

- [ ] 完整的剧本流程可正常运行
- [ ] 世界地图和场景切换正常工作
- [ ] 存档/读档功能完整
- [ ] 专长和背景特性正确应用（B304/B305，若 v0.3 B203 状态效果未完成则在此版本补齐）
- [ ] LLM 上下文压缩正常工作

## 5. 验收测试场景

### 场景 1：剧本完整流程

1. 用户选择剧本开始游戏
2. 完成第一章任务
3. 解锁第二章
4. 推进剧情直到完成
5. 显示完成信息

### 场景 2：世界地图旅行

1. 用户打开世界地图
2. 查看可用地点
3. 选择目标地点
4. 开始旅行
5. 可能触发随机遭遇
6. 到达目的地

### 场景 3：存档管理

1. 用户创建存档
2. 继续游戏
3. 创建另一个存档
4. 加载第一个存档
5. 验证状态正确恢复
6. 删除不需要的存档
