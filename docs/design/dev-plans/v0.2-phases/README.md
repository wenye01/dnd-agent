# v0.2 开发计划 - 分阶段执行

> **版本**: v0.2
> **目标**: 角色创建 + 基础 UI
> **总时长**: 约 9 天

---

## 概述

v0.2 版本将 v0.1 的基础设施扩展为可用的游戏原型，用户可以：
- 通过自然语言创建 D&D 角色
- 与 AI DM 进行对话
- 在界面中查看角色信息
