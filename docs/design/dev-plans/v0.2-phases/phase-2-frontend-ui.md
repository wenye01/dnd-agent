### F008 - 主界面布局

| 属性 | 描述 |
|------|------|
| **任务ID** | F008 |
| **任务名称** | 主界面布局实现 |
| **所属模块** | components/layout |
| **优先级** | P0 |

**功能描述**：
- 实现游戏主界面布局（三栏式）
- 实现左侧面板区（角色、队伍）
- 实现中间主区域（聊天、地图）
- 实现右侧操作区（快捷操作）

**布局结构**：
```
┌─────────────────────────────────────────────────────────┐
│                         Header                          │
├──────────────┬────────────────────────┬────────────────┤
│              │                        │                │
│  左侧面板    │      中间主区域         │   右侧操作区   │
│  (角色)      │      (聊天/地图)        │   (快捷操作)   │
│              │                        │                │
└──────────────┴────────────────────────┴────────────────┘
```

**验收标准**：
1. 布局响应式适配
2. 各区域可以正确显示内容
3. 布局可以自定义调整

**详细指南**: 参考 `v0.2-frontend.md` 迭代 0.2.3
