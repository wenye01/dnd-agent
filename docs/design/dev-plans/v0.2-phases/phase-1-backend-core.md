> **版本**: v0.2

> **预计时间**: 第1-3天

## 阶段目标

完成后端核心功能：
1. 角色系统基础（3种族/职业原型）

## 任务清单

### B004 - 角色系统基础

| 属性 | 描述 |
|------|------|
| **任务ID** | B004 |
| **任务名称** | 角色系统基础实现 |
| **所属模块** | server/character |
| **优先级** | P0 |

**功能描述**：
- 定义角色数据模型（Character, NPC, Enemy）
- 实现角色创建（种族、职业、背景、属性）
- 实现属性计算（属性修正、熟练加值、技能加值）
- 实现基础属性（HP、AC、速度、先攻）

**验收标准**：
1. 支持常用种族（人类、精灵、矮人）和职业（战士、法师、盗贼）作为原型验证
2. 属性计算符合 D&D 5e 规则（修正符、熟练加值、技能加值）
3. 角色可正确序列化为 JSON
4. 单元测试覆盖核心计算逻辑

**详细指南**: 参考 `v0.2-backend.md` 迭代 0.2.1

### B007 - Server 工具注册

| 属性 | 描述 |
|------|------|
| **任务ID** | B007 |
| **任务名称** | Server 工具注册和执行 |
| **所属模块** | client/tools |
| **优先级** | P0 |

**功能描述**：
- 实现工具注册表
- 定义核心工具（roll_dice, create_character, get_character 等）
- 实现工具执行和结果返回
- 实现工具 Schema 定义（JSON Schema）

**必须实现的 5 个核心工具**：
1. `roll_dice` - 骰子投掷
2. `create_character` - 创建角色
3. `get_character` - 获取角色信息
4. `get_game_state` - 获取游戏状态
5. `roll_ability_check` - 属性检定

**验收标准**：
1. 工具 Schema 符合 OpenAI Function Calling 格式
2. 工具执行结果可以正确序列化
3. 错误可以正确返回给 LLM

**详细指南**: 参考 `v0.2-backend.md` 迭代 0.2.2
