# 开发计划 - v0.1 基础设施

> **版本**: v0.1.0
> **里程碑**: 基础设施骨架
> **创建日期**: 2026-03-18

---

## 1. 版本概述

### 1.1 版本目标

搭建项目骨架，建立前后端通信基础，集成 LLM（OpenAI），实现持久化和 WebSocket 通信。

### 1.2 核心价值

- **基础设施**：可运行的前后端骨架
- **LLM 集成**：OpenAI 适配层，支持工具调用
- **通信层**：WebSocket + REST API
- **持久化**：游戏状态可保存和恢复

### 1.3 发布标准

- [ ] 后端可启动，WebSocket 可连接
- [ ] LLM 流式叙述正常工作
- [ ] 状态可持久化并恢复
- [ ] 前端可连接后端（WebSocket 客户端 + 状态管理）

| **所属模块** | server/dice |
| **优先级** | P0 |

**功能描述**：
- 实现骰子表达式解析器（支持 d20, 2d6, 4d6k3 等）
- 实现掷骰器（支持普通、优势、劣势）

**实现目的**：

**预期效果**：

```go
result := roller.Roll() // {Dice: [4,5], Modifier: 3, Total: 12}

// 属性检定
check := dice.AbilityCheck(character, AbilityDex, 15, false)
// {Success: true, Roll: 18, Modifier: 3, DC: 15}
```

**验收标准**：
1. 支持标准骰子表达式（NdS, NdSkM, +/-修饰符）
2. 掷骰结果分布符合预期
3. 属性修正符计算正确
4. 单元测试覆盖率 > 80%
