## v1 实现范围摘要

### 实现的功能

- 完整的 D&D 5e 游戏流程（角色创建、剧本选择、探险、战斗）
- 多 LLM 提供商支持（OpenAI、Anthropic）
- Web 前端界面（React + Phaser）
- 战棋式战斗系统
- 多存档系统

### 不实现的功能（后续版本）

- 多用户联机
- 用户自定义剧本创建
- 移动端支持
- 惊奇轮（Surprise Round）
- 多职业系统
- 光照对战斗的影响
- 剧本解析工具（当前使用手动预处理）

## 关键技术决策

| 决策项 | 选择 |
| --- | --- |
| **后端语言** | Go |
| **前端技术栈** | React + Phaser |
| **通信协议** | WebSocket |
| **地图格式** | FVTT |
| **内部通信** | MCP 格式（合并部署） |
| **多角色控制** | 轮流控制模式 |
| **战斗触发** | 混合模式（触发器 + LLM） |
| **角色死亡** | 允许复活 |
